# 📚 LUMINA Library Management System

**Live Production Link:** [https://lumina-da6v80tt5-vimals-projects-e2b5a73c.vercel.app](https://lumina-da6v80tt5-vimals-projects-e2b5a73c.vercel.app)

LUMINA-LMS is a modern, full-stack library management system designed for seamless book circulation, real-time tracking, and AI-powered member assistance.

## 🚀 Key Features
- **Admin Dashboard**: Comprehensive analytics on books, members, and circulation.
- **Visual Rack Mapping**: Interactive rack layouts for easy book location.
- **AI Assistant**: Groq-powered AI for book recommendations and queries.
- **Smart Search**: Fast, filterable search for books and members.
- **Responsive UI**: Built with React and Tailwind CSS for a premium look.

## 🛠️ Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Lucide Icons.
- **Backend**: Node.js, Express.
- **Database**: PostgreSQL (pg).
- **AI Engine**: Groq SDK & ElevenLabs.

## 📦 Local Setup
1. **Clone the repo**:
   ```bash
   git clone https://github.com/vimalvijay06/LUMINA-LMS.git
   cd LUMINA-LMS
   ```
2. **Install dependencies**:
   ```bash
   npm install
   cd backend && npm install
   ```
3. **Environment Variables**:
   Create a `.env` in the root and `backend/` directories with your credentials.
4. **Run the app**:
   ```bash
   npm run dev
   ```

## 🌐 Deployment
This project is configured for one-click deployment on **Render** (Backend) and **Vercel** (Frontend). See `deployment_walkthrough.md` for details.
